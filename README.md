# 정우배틀

Mobile web arena shooter test game.

## Play

After GitHub Pages deploys:

`https://kjuws10-rgb.github.io/JeongwooBattle/`

On iPhone Safari, use Share -> Add to Home Screen to run it like an app.

## Controls

- Left joystick: move
- Right button: shoot
- Collect blue cubes for stronger shots
- Collect orange missile items for burst shots
- Collect blue shield items for temporary protection
- Survive waves and defeat enemies

The arena includes solid obstacles, sound effects, and background music. Audio starts after the first touch or key input.
